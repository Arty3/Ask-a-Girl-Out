## 🛠 Requirements

- [Nodejs](https://nodejs.org/en)

## 🚀 Installation

1. Clone the repository:

```sh
git clone https://github.com/DontCallMeLuca/Ask-a-Girl-Out
cd Ask-a-Girl-Out
```

2. Install dependencies:

```sh
npm install framer-motion
```

3. Run

```sh
npm run dev
```
